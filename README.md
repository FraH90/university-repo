# University Projects Archive

Collection of selected university projects in electronics engineering, RF/microwave design, digital communications, analog simulation, and Python-based technical computing.

This repository groups together older course projects, lab work, and thesis material into a single archive so they are easier to browse and maintain. Each subfolder remains self-contained and keeps its own documentation, source files, simulations, and supporting material.

## Projects

### [B.Sc. Thesis: Multipaction Effect in S-Band Components](./bsc-thesis-multipaction-effect/README.md)
Study of the multipaction effect in waveguide-based RF components for particle accelerators, including the thesis manuscript and discussion presentation.

### [Microwave Transformer Comparison](./mwave-filter-comparison/README.md)
Python simulations comparing binomial and Chebyshev impedance transformers, with plots of reflection coefficient and bandwidth tradeoffs.

### [Phased Array Antennas](./phased-array-antennas/README.md)
MATLAB and Antenna Toolbox project focused on phased-array beam steering, supported by a short report and animation outputs.

### [Gm-C Voltage-Controlled Filters](./gm-c-filters/README.md)
OrCAD/PSpice design and simulation of first- and second-order Gm-C filters, exploring OTA transconductance control and quality factor variation.

### [Hamming TX/RX](./hamming-tx-rx/README.md)
Simulink implementation of a Hamming-code transmitter/receiver chain with single-bit error injection, detection, and correction.

### [6T SRAM Cell](./sram-cell-6T/README.md)
OrCAD Capture and PSpice simulation of a 6-transistor SRAM cell, with material covering read and write behavior.

### [Electronic Design Automation Notes](./electronic-design-automation-notes/README.md)
EDA laboratory notes and circuit-design exercises developed with OrCAD Capture and PSpice.

## Notes

- The repository is intended as a curated archive rather than a single unified software project.
- Some projects were originally developed with student editions of EDA tools or university toolchains.
- Large presentation material is tracked with Git LFS where needed.
